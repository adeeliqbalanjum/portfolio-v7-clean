# Portfolio v7 Clean

Clean static portfolio for Muhammad Adeel Iqbal.

## Stack

- Static HTML
- CSS
- Small vanilla JavaScript
- No npm dependencies
- No build step
- GitHub Pages via GitHub Actions

## Deploy

Go to **Settings → Pages → Source → GitHub Actions**. Push to `main`. The included workflow deploys the site.

Live URL after deployment:

```txt
https://adeeliqbalanjum.github.io/portfolio-v7-clean/
```

## Edit content

- Homepage: `index.html`
- Portfolio listing: `portfolio/index.html`
- Styles: `assets/css/styles.css`
- Interactions: `assets/js/site.js`
- SEO: update canonical/meta tags, `sitemap.xml`, and `robots.txt`

## Notes

This repo intentionally avoids React, Next.js, heavy animation libraries, patch CSS, and build tooling so the portfolio stays fast, simple, and reliable on GitHub Pages.
